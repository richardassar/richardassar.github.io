Projects

  • Omega Engine

    A multiparadigm game engine breaking away from the bug-prone approaches commonplace in the games industry. Leveraging pattern matching, term and graph rewriting, program synthesis and scheduling - coupled with axiomatic models of hardware and APIs. More than just a game engine this project establishes a foundation for all scientific computing, machine learning research and creative endeavours of mine going forward.

  • Multichannel Signed Distance Field Fonts for VR

    Inspired by the MSDFGEN thesis project I applied differentiable rendering, incorporating mipmapping and anisotropic filtering, to learn compact multi-channel median filtered signed distance field textures for sharper and more efficient GPU text rendering in VR.

  • 3D IFS fractal renderer

    Implemented in the Omega Engine using Vulkan compute shaders this applies OIT (order-independent transparency) so that non-commutative alpha blending functions can be utilised. The resulting images can take on many forms but so far the aesthetics of galaxies or smoke effects are easy to achieve with animated attractors.

  • C++ serialization tool

    Serialization code is auto-generated using memory layout information computed by Clang during compilation allowing complex data types and pointer graphs to be serialized in a highly performant manner with minimal impact on code. Simply flag the classes/structs, with no restrictions on polymorphism or virtual inheritance, that you wish to be serializable and then stream them into and out of any storage buffer or stream.

  • Wolfram Language Evaluator

    Inspired by the Physics Project and seeking deeper understanding of the construction of a CAS (computer algebra system), implemented in C++ and using LLVM for JIT and compilation. Based on the description of the evaluator from "Power Programming with Mathematica: The Kernel".

  • Neural DSP based Piano Synthesizer

    The culmination of about 4 years of research, after several different approaches. This differentiable DSP synthesiser can replicate physical instruments using neural models inductively biased with additive and subtractive synthesis and a learned convolution reverb. The final model produces CD quality audio using only a small fraction of a single core.

  • OmegaOS

    A homebrew operating system that boots into a lambda calculus interpreter, which itself is written in BLC (binary lambda calculus).

  • SampleRNN_torch

    An unconditional neural generative model of raw audio implemented in the Torch framework, capable of modelling complex audio at the sample level. I've many future research plans involving conditional variants of this model applied in a variety of settings.

  • VisionMachine

    An LLVM powered, gesture driven, flow-based programming compiler and development environment. This project aims to circumvent the limitations of traditional compilers and programming languages, balancing creativity with optimal machine code generation, and the ability to hot-patch changes live several orders of magnitude faster than other solutions (Edit and Continue, Indefiant’s Recode). Currently in a prototypical phase, future project aims include GPU acceleration via technologies such as KernelGen and Polly. With time this could be a strong competitor against products such as MaxMSP, PureData and LabView by supporting heterogeneous computing architectures transparently.

  • Near-Eye Lightfield Display

    Recreating the VR glasses as described in Douglas Lanman's paper, Near-Eye Light Field Displays. This project is a great opportunity to get hands-on experience with microlenses and microdisplays, solid state gyroscope and lightfield rendering. I have implemented rasterisation and path tracing lightfield renderers, using a combination of OpenGL and OpenCL. The next phase is to house the glasses and mount the solid-state motion tracker. Studying the paper was a good learning experience in the optics and mathematics involved.

  • Bitcoin Payment Processing System

    With a custom fork of the bitcoin-core client, adding ZeroMQ support, RabbitMQ, PostgreSQL, Node.js and custom frontend JavaScript users can deposit and withdraw funds via QR code - receiving realtime notifications via WebSocket. The IP for this project is held by my company CoinTech Ltd.

  • Projector-Depthcam Calibration Software

    Performs pose estimation between a depth camera and projector, creating an RGBD video feed as seen from the projector-eye-view. This has applications in interactive installations.
